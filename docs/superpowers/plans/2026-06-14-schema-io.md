## Task 3 — Converter: Dart → TS + links both ways

**Files**: `lib/src/schema/instant_schema_io.dart`, `test/schema_io_test.dart`.

### 3a. Failing tests first
```dart
test('emits instant.schema.ts from Dart models', () {
  final ts = emitInstantTs(parseDartModels(dartModelsSrc));
  expect(ts, contains("import { i } from '@instantdb/react'"));
  expect(ts, contains('todos: i.entity({'));
  expect(ts, contains('i.boolean()'));
  expect(ts, contains('i.string().unique()')); // id
  expect(ts, contains('export default schema'));
});

test('@InstantField(unique/indexed) round-trips to modifiers', () {
  final ts = emitInstantTs(parseDartModels(
    "@InstantModel('users') class User { final String id; "
    "@InstantField('email', unique: true, indexed: true) final String email; "
    "const User({required this.id, required this.email}); }"));
  expect(ts, contains('i.string().unique().indexed()'));
});

test('links: forward/reverse <-> @InstantLink both sides', () {
  final s = parseInstantTs(linkTs);     // todos.owner <-> users.todos
  final dart = emitDart(s);
  expect(dart, contains('@InstantLink()'));
  expect(dart, contains('User? owner'));      // to-one
  expect(dart, contains('List<Todo> todos')); // to-many
  final ts2 = emitInstantTs(parseDartModels(dart));
  expect(ts2, contains('links: {'));
  expect(ts2, contains("has: 'one'"));
  expect(ts2, contains("has: 'many'"));
});

test('system-entity link emits only the user-side @InstantLink', () {
  // todos.author -> $users : Dart has Todo.author (User?), no $users class
  final dart = emitDart(parseInstantTs(sysLinkTs));
  expect(dart, contains('User? author'));
  expect(dart, isNot(contains('class User')));
});

test('round-trip stability TS->Dart->TS', () {
  final ts1 = todosTs;
  final ts2 = emitInstantTs(parseDartModels(emitDart(parseInstantTs(ts1))));
  // entities + field types + modifiers preserved (number->num documented)
  expect(parseInstantTs(ts2).entities.map((e) => e.name),
      containsAll(parseInstantTs(ts1).entities.where((e)=>!e.system).map((e)=>e.name)));
});
```

### 3b. Implement
- `parseDartModels`: regex/scan for `@InstantModel('ns')` + class body; read
  `final <type> <name>;`, `@InstantField(...)`, `@InstantLink(...)`; Dart→instant
  type map; nullable → optional; `id`→unique. Build `SchemaDef` incl. `LinkDef`s
  from paired `@InstantLink` fields.
- `emitDart` link emission (complete Task-2 stub): per `LinkDef`, add the
  forward `@InstantLink` field on `fromEntity`'s class and the reverse on
  `toEntity`'s class; skip the side landing on a system entity; cardinality →
  `T?` / `List<T>`.
- `emitInstantTs`: entities + `links` object; dedupe reciprocal links; synthesize
  reverse when only one side declared; re-emit system entities only if present in
  a passed-through original (optional param).

Run `flutter test test/schema_io_test.dart` → all pass. `flutter analyze` clean.
